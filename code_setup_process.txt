conda create -p ./venv python=3.8 -y                   ---------->> for better approch use this - This explicitly creates the environment in the current project directory.
                { conda info                            ----> if python version get some error do check the version

                conda create -p ./venv python==3.8 -y }

conda create -n ds_project python=3.8 -y          -----> Instead of creating an environment by path (-p), create it by name:
